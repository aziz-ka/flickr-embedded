# [Flickr Embedded](https://internal-eyes.surge.sh)


## Stack

- React
- Socket.io
- Bootstrap 4


#### Quick Start

```shell
$ git clone git@gitlab.com:flickr-amex/flickr-embedded.git
$ cd flickr-embedded
$ yarn
$ yarn start
```


#### Yarn/NPM Commands

|Script|Description|
|---|---|
|`yarn start`|Start the app on `localhost:3000`|
|`yarn build`|Build the app to `./build`|
